## Project Configuration

- **Language**: TypeScript
- **Package Manager**: bun
- **Add-ons**: prettier, sveltekit-adapter, devtools-json, eslint

Plex official API: https://developer.plex.tv/pms/
(To access library api documentation for example use https://developer.plex.tv/pms/#tag/Library - so /pms/#tag/{thing you want})

Endpoints in the Official API:

- General
- Library
- Library: Playlists
- Library: Collections
- Status
- Activities
- Updater
- Butler
- Events
- Log
- Preferences
- Download Queue
- UltraBlur
- Transcoder
- Provider
- Metadata Agents
- Content
- Hubs
- Search
- Rate
- Playlist
- Play Queue
- Timeline


Plex Unofficial Python API which has undocumented API endpoints: https://python-plexapi.readthedocs.io/en/latest/index.html
